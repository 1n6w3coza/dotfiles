# Installation
```
cd
git clone https://github.com/flipsidecreations/dotfiles.git
ln -s dotfiles/.bashrc ~/
ln -s dotfiles/.vimrc ~/
ln -s dotfiles/.screenrc ~/
ln -s dotfiles/.tmux.conf ~/
```
